# 🛒 E-commerce Platform  |  Rugal Modas

Plataforma de e-commerce desenvolvida para uma empresa com o objetivo de expandir sua presença digital, oferecendo uma experiência de compra moderna, segura e escalável.

## 📌 Visão Geral

A aplicação foi projetada para atender tanto clientes quanto administradores, centralizando funcionalidades essenciais de uma loja virtual em um único sistema.

### O projeto prioriza:

Arquitetura escalável
Performance
Organização de código
Experiência do usuário (UX)
---

### 🚀 Tecnologias

## 🧠 Arquitetura do Projeto
src/
 ├── assets/         # Recursos estáticos
 ├── components/     # Componentes reutilizáveis
 ├── pages/          # Páginas principais
 ├── services/       # Camada de serviços (API, lógica externa)
 ├── hooks/          # Hooks personalizados
 ├── styles/         # Estilos globais
 └── App.jsx         # Entry point da aplicação

## ▶️ Execução do Projeto
### Clone o repositório
git clone https://github.com/seu-usuario/seu-repositorio.git

### Acesse o diretório
cd seu-repositorio

### Instale as dependências
npm install

### Execute em ambiente de desenvolvimento
npm run dev

A aplicação estará disponível em:

http://localhost:5173
📦 Build e Deploy
# Gerar build de produção
npm run build

# Visualizar build local
npm run preview
🔐 Segurança e Acesso

O sistema implementa controle de acesso baseado em perfis:

Usuários: acesso às funcionalidades de navegação e compra
Administradores: acesso ao dashboard e gerenciamento da plataforma

Estrutura preparada para futura integração com autenticação robusta (JWT, OAuth, etc.)
